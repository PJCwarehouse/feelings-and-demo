git status 就可以看你当前和远端仓库的差别，你这个命令行不支持中文 等会去网上查教程弄一下

git add . 就是自动把修改添加到 commit 列表

现在看git status 就可以看到他识别到 ：新增了一个文件

git commit -m"提交信息" 提交（本地仓库）这一步执行完会提交到本地仓库并产生一条记录

同步到远端仓库：git push
